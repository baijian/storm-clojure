# storm-clojure

storm demo using clojure

## Installation

install [leiningem](https://github.com/technomancy/leiningen) first.

$ git clone git@github.com:baijian/storm-clojure

## Usage

storm-clojure: demo

    $ java -jar storm-clojure-0.1.0-standalone.jar [args]
